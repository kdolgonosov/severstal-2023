# Тестовое задание "Заметки" для стажёрской программы IT HUB "Северстали"

## Для запуска:
```
npm i
npm run start
```

## Стек:

-   React
- Material UI

## Функциональность:

-   создание, редактирование, удаление заметок
-   сохранение заметок пользователя в localStorage
-   настраиваемый поиск по заметкам
